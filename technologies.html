ONZE AANPAK: de pragmatische

Een succesvol project is in de eerste plaats handige functies creëren voor de gebruikers ervan. Technologie moet beschikbaar zijn voor het project en niet omgekeerd.

Uw project is geen R & D lab of een leerperiode voor onze ontwikkelaars. We maken gebruik van technologieën die we al bewezen hebben.

WEB & MULTI-PLATFORM

 

In onze business evolueert technologie snel en uw teams hebben moeite om bij te blijven.

Maar technologie kan een beslissende factor zijn in de strijd tegen de concurrentie. Wij zijn actief in verschillende technologische gemeenschappen ook in de verschillende programmeertalen: Java, Scala, robijn, javascript en backend services.

CLOUD

 

De wolk biedt flexibiliteit en boost het opstarten van uw projecten. We kennen ook de grenzen. Rust in 2012, de cloud is een essentieel onderdeel van veel projecten.

We gebruiken de cloud om applicaties in te zetten, om tests uit te voeren, of voor onze samenwerking met CloudBees, software factory voor onze projecten.

De wolk is een onderdeel van Xebia Studio-projecten en we werken met de grote industrie leiders: Amazon Web Services, Heroku, CloudBees, OVH.

NoSQL : NOT ONLY SQL

 

NoSQL technologieën breken met de traditionele databases die we al decennia gebruiken. We hebben verschillende projecten van dit type base voorzien. (MongoDB, Redis, HBase / Hadoop).

NoSQL kan een geweldige "boost" voor het project betekenen. De ontwikkelsnelheid en implementatietijd (door het ontbreken van bindende patronen) is een belangrijke een troef voor onze projecten. We weten ook wanneer NoSQL niet gebruikt moet worden of wanneer deze technologie gebruikt moet worden met de traditionele relationele databases.
