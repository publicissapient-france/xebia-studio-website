ONZE AANPAK: de pragmatische

Een succesvol project is in de eerste plaats zinvolle functies creëren voor de gebruikers ervan. Technologie moet beschikbaar zijn voor het project en niet omgekeerd.

Uw project is geen R & D lab of een leerperiode voor onze ontwikkelaars. We maken gebruik van technologieën die we al bewezen hebben.

WEB & MULTIPLATFORM

 

In onze business evolueert technologie snel en uw teams hebben moeite om bij te blijven.

Maar technologie kan een beslissende factor zijn in de strijd tegen de concurrentie. Wij zijn actief in verschillende technologische gemeenschappen en hebben kennis van verschillende programmeertalen: Java, Scala en Ruby.

CLOUD

 

De cloud biedt flexibiliteit en uw projecten krijgen hiermee een vliegende start.

We gebruiken de cloud in onze projecten om applicaties in te zetten, om tests uit te voeren, of voor onze samenwerking met CloudBees.

De cloud is een onderdeel van Xebia Studio-projecten en we werken met de grote industrie leiders: Amazon Web Services, Heroku, CloudBees, OVH.

NoSQL : NOT ONLY SQL

 

NoSQL technologieën breken met de traditionele databases die we al decennia gebruiken. We hebben verschillende projecten van dit type base voorzien. (MongoDB, Redis, HBase / Hadoop).

NoSQL kan een geweldige "boost" voor het project betekenen. De ontwikkelsnelheid en implementatietijd (door het ontbreken van bindende patronen) is een belangrijke troef voor onze projecten. We weten ook wanneer NoSQL niet gebruikt moet worden of wanneer deze technologie gebruikt moet worden met de traditionele relationele databases.
