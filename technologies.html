ONZE AANPAK: de pragmatische

Een succesvol project bestaat in de eerste plaats om waarde te leveren voor de gebruikers ervan. Technologie moet ten dienste staan van het project en niet omgekeerd.

Uw project is geen R & D lab of een leerperiode voor onze ontwikkelaars. We maken gebruik van technologieën die we al bewezen hebben, en die we in overleg met u selecteren.

WEB & MULTI-PLATFORM

 

In onze business evolueert technologie snel en uw teams kunnen moeite hebben om bij te blijven.

Maar technologie kan een beslissende factor zijn in de strijd tegen de concurrentie. Wij zijn actief in verschillende technologische gemeenschappen, ook in verschillende programmeertalen: Java, Scala, Clojure, Objective C, javascript en backend services.

CLOUD

 

De cloud biedt flexibiliteit en boost het opstarten van uw projecten. We kennen ook de grenzen. Niettemin is de cloud inmiddels een essentieel onderdeel van veel projecten.

We gebruiken de cloud om applicaties in te zetten, om tests uit te voeren, of voor onze samenwerking met CloudBees.

De Cloud is een onderdeel van Xebia Studio-projecten, en we werken daarin met de grote industrie leaders: Amazon Web Services, Heroku, CloudBees, OVH.

NoSQL : NOT ONLY SQL

 

NoSQL technologieën breken met de traditionele databases die we al decennia gebruiken. We hebben verschillende projecten van dit type database uitgevoerd en begeleiden diverse implementaties bij klanten. (MongoDB, Redis, HBase / Hadoop).

NoSQL kan een geweldige "boost" voor het project betekenen. De ontwikkelsnelheid en implementatietijd (door het ontbreken van bindende schema's) is een belangrijke troef voor onze projecten. We weten ook wanneer NoSQL niet gebruikt moet worden of wanneer deze technologie gebruikt moet worden in combinatie met traditionele relationele databases.
