CLOUD

 

De wolk biedt flexibiliteit en boost het opstarten van uw projecten. We kennen ook de grenzen. Rust in 2012, de cloud is een essentieel onderdeel van veel projecten.

We gebruiken de cloud om applicaties in te zetten, "gevallen", aldus de belasting toe te voegen, om tests uit te voeren, of gewoon met name door onze samenwerking met CloudBees, software factory voor onze projecten.

De wolk is een onderdeel Xebia Studio-projecten, en we werken met de industrie leiders: Amazon Web Services, Heroku, CloudBees, OVH.
